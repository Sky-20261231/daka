# 打卡 — 使用说明

一个运行在手机上的个人习惯追踪工具，支持每日打卡提醒和阶段性复盘。

---

## 一、如何安装

### 安卓手机（推荐方式）

1. 打开手机浏览器（推荐 Chrome 或系统自带浏览器）
2. 在地址栏输入这个地址并打开：

   👉 **https://sky-20261231.github.io/daka/**

3. 页面加载后会弹出提示"**添加到主屏幕**"，点击确认就可以了
4. 如果没有弹出提示，点击浏览器右上角 `⋮` → 选择"添加到主屏幕"
5. 桌面上会出现「**打卡**」图标，点击就能像普通 App 一样使用了

### iPhone

1. 用 **Safari** 打开上述地址
2. 点击底部中间的 **分享按钮**（方框带箭头）
3. 滑动找到「**添加到主屏幕**」→ 点右上角"添加"
4. 桌面出现「打卡」图标 → 完成

---

## 二、添加习惯

打开应用后，点击右下角的蓝色 **+** 按钮，进入新增习惯界面：

| 字段 | 说明 |
|------|------|
| 名称 | 你想追踪的习惯，比如"晨间阅读""喝够8杯水"（最多20个字） |
| 类型 | 见下方说明 |
| 颜色 | 给习惯选一个颜色标签，方便区分 |

### 两种类型

**☑ 完成型** — 做了就是做了，比如"早起""冥想""读书"

- 点击习惯卡片 = 完成 ✓
- 再点一下 = 取消
- 适合记录"做没做"

**📊 数值型** — 需要记录具体数字，比如"喝水""运动""阅读时长"

- 点击习惯卡片，输入数字后点"记录"
- 适合记录"做了多少"

---

## 三、每日打卡

打开应用默认进入「今日」页面，顶部显示当前日期。

- 如果是**完成型**习惯 → 直接点击习惯卡片，打勾即完成
- 如果是**数值型**习惯 → 点击弹出输入框，输入数字后点「记录」

### 补卡（补打前几天的卡）

日期栏左右两边的 **◀ ▶** 按钮可以切换日期，往前翻到要补卡的日期，正常打卡即可。

切换到的日期如果不是今天，页面顶部会出现一行黄色提示"当前查看的是过去的日期"，表示正在补卡。

---

## 四、复盘

底部中间「**复盘**」标签，可以查看每个习惯的统计数据：

| 功能 | 说明 |
|------|------|
| 顶部标签栏 | 点击切换要复盘的习惯 |
| 统计分析 | 完成型：总天数、连续天数、完成率 / 数值型：每周总量、每月总量、单日最高、同比变化 |
| 图表 | 8 周趋势柱状图、6 月趋势柱状图、工作日分布图、30 天折线图 |
| 日历 | 按月份展示每天的数据值，颜色越深数值越大 |

---

## 五、设置提醒

底部右边「**设置**」标签：

1. **管理习惯**：点击习惯右边的 ✏️ 可以修改名称、类型、颜色，也可以删除
2. **每日提醒时间**：设置每天几点推送打卡提醒

   > 如果是浏览器打开的（没安装到桌面），提醒可能不生效。建议安装到桌面后再用提醒功能。

3. **通知权限**：首次使用会提示开启通知权限，点击"开启通知权限"按钮即可

## 六、常见问题

### 断网能用吗？

可以。安装到桌面后，即使没网也能正常打卡，数据都在手机本地。联网后会加载最新版本（如果有更新）。

### 我的数据会传到网上吗？

不会。所有打卡记录只存在你手机的浏览器里，不会上传到任何服务器。别人无法看到你的数据。

### 分享给别人用，别人能看到我的打卡吗？

看不到。每个人打开这个链接相当于一份独立的副本，数据互不相通。

### 怎么更新到最新版本？

打开应用时会自动检测新版本，刷新一次页面即可更新。如果长时间看不到更新，在浏览器中手动刷新一下。

### 换个手机数据能迁移吗？

目前不支持跨设备同步。如果需要这个功能可以联系开发者。
